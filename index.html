Markdown Syntax and HTML Equivalents

Markdown Syntax Description HTML Equivalent
# Heading 1 Creates a top-level heading <h1>Heading 1</h1>
## Heading 2 Creates a second-level heading <h2>Heading 2</h2>
**bold text** Makes text bold <strong>bold text</strong>
*italic text* Makes text italic <em>italic text</em>
- List item Creates a bullet point list <ul><li>List item</li></ul>
1. List item Creates an ordered list <ol><li>List item</li></ol>
[Link Text](http://...) Creates a hyperlink <a href="http://...">Link Text</a>
![Image](image.jpg) Embeds an image <img src="image.jpg" />
> Blockquote Creates a blockquote <blockquote>Blockquote</blockquote>
`` Inline code `` Displays inline code <code>Inline code</code>
``` Code block ``` Displays a code block <pre><code>Code block</code></pre>
